# Website

基于 VuePress 2.0 和 Theme Hope 构建的个人知识库网站。

## 📖 项目简介

本项目是一个用于记录和整理技术学习笔记、算法刷题记录以及数学知识的文档站点。通过现代化的文档工具，打造结构清晰、易于检索的个人知识体系。

## 🛠️ 技术栈

- **框架**: VuePress 2.0
- **主题**: vuepress-theme-hope 2.0
- **构建工具**: Webpack
- **编程语言**: TypeScript, Vue 3
- **样式**: Sass
- **公式渲染**: KaTeX
- **图表绘制**: Mermaid
