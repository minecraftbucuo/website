# 初等方法证明巴塞尔问题（2026.4.25）

如果你曾经接触过微积分或无穷级数，你大概率听说过大名鼎鼎的**巴塞尔问题 (The Basel Problem)**。这个问题要求计算所有非零自然数平方的倒数和：

$$\sum_{n=1}^{\infty} \frac{1}{n^2} = 1 + \frac{1}{4} + \frac{1}{9} + \frac{1}{16} + \cdots$$

这个问题曾困扰了当时的数学界近半个世纪，直到 1734 年被年轻的莱昂哈德·欧拉 (Leonhard Euler) 绝妙地解出，答案是惊人的 $\frac{\pi^2}{6}$。欧拉最初的证明充满了天才的直觉，但在当时缺乏严密的收敛性证明（他直接将泰勒展开式与无穷乘积画上了等号）。

今天，我们要学习的是由 Ioannis Papadimitriou 在 1973 年发表的一种**严密的初等证明方法**。它不需要复分析或傅里叶级数等高深工具，只需要你掌握**高中级别的三角函数、复数基础（二项式定理）和多项式韦达定理**。

让我们一步步揭开这个精美证明的面纱。

---

## 第一步：从一个意想不到的角度切入 —— 复数与三角函数展开

证明的第一步，看似和无穷级数毫无关系。我们需要寻找 $\sin(n\theta)$ 与 $\cot\theta$ 之间的代数关系。

这里我们需要请出连接复数与三角函数的桥梁——**棣莫弗公式 (De Moivre's formula)**：
$$(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$$

接下来，我们利用**二项式定理**将等式的左边 $( \cos\theta + i\sin\theta )^n$ 展开。
二项式定理的形式是 $(a+b)^n = \sum_{k=0}^n \binom{n}{k} a^{n-k}b^k$。在这里 $a = \cos\theta$，$b = i\sin\theta$。

由于等式右边实部是 $\cos(n\theta)$，虚部是 $\sin(n\theta)$，我们**只关心等式两边的虚部**（即所有包含奇数个 $i$ 的项，因为 $i^1 = i$，$i^2 = -1$（实数），$i^3 = -i$，$i^4 = 1$（实数），以此类推，符号会交替出现）：

提取虚部后，我们就得到了 $\sin(n\theta)$ 的表达式，注意其中的正负号交替：
$$\sin(n\theta) = \binom{n}{1}\cos^{n-1}\theta \sin\theta - \binom{n}{3}\cos^{n-3}\theta \sin^3\theta + \binom{n}{5}\cos^{n-5}\theta \sin^5\theta - \cdots$$

为了让这个式子变得更整齐，并且引出我们想要的 $\cot\theta$，我们**在等式右侧强行提取一个公因式 $\sin^n\theta$**。
我们拿第二项举例看看提取后会变成什么样：
$$\frac{\cos^{n-3}\theta \sin^3\theta}{\sin^n\theta} = \frac{\cos^{n-3}\theta}{\sin^{n-3}\theta} = \left(\frac{\cos\theta}{\sin\theta}\right)^{n-3} = \cot^{n-3}\theta$$

按照这个规律，所有项都可以完美转化为 $\cot\theta$ 的幂，式子变为了一个极其整齐的乘积形式：

$$\sin(n\theta) = \sin^n\theta \left[ \binom{n}{1}\cot^{n-1}\theta - \binom{n}{3}\cot^{n-3}\theta + \binom{n}{5}\cot^{n-5}\theta - \cdots \right]$$

这就是我们后续所有推导的基石。

---

## 第二步：巧妙构造方程，召唤“韦达定理”

现在，真正的魔法开始了。我们要把上面的三角恒等式变成一个标准的多项式方程。

1. **设定特殊参数，确保最高次幂为偶数**：
   我们令 $n = 2m + 1$（其中 $m$ 为正整数）。这样 $n-1 = 2m$，括号里的最高次幂就是 $\cot^{2m}\theta$，恰好是偶数次幂，方便我们稍后换元。

2. **寻找方程的零点（根）**：
   我们要让等式左边的 $\sin((2m+1)\theta) = 0$。
   我们知道，当角度是 $\pi$ 的整数倍时，正弦值为 0。即 $(2m+1)\theta = k\pi$（$k$ 为整数）。
   由此，我们可以找到 $m$ 个位于区间 $(0, \pi/2)$ 内的角度：
   $$\theta_k = \frac{k\pi}{2m+1} \quad (k = 1, 2, \dots, m)$$
   
   **关键逻辑来了：** 在 $(0, \pi/2)$ 这个区间内，$\sin(\theta_k)$ 绝对大于 0。回顾我们第一步得到的公式：$\text{左边} = \sin^n\theta \times [\text{方括号}]$。
   既然整体等于 0，且 $\sin^n\theta_k \neq 0$，那么**唯一的可能就是方括号里的那串式子等于 0**！

3. **构建 $m$ 次多项式并换元**：
   我们将方括号内的式子看作是关于新变量 $x$ 的多项式，令 $x = \cot^2\theta$。
   因为 $\cot^{n-1}\theta = \cot^{2m}\theta = (\cot^2\theta)^m = x^m$，同样 $\cot^{n-3}\theta = x^{m-1}$，依此类推。多项式方程变为：
   $$P(x) = \binom{2m+1}{1}x^m - \binom{2m+1}{3}x^{m-1} + \dots = 0$$
   
   我们刚才找到了 $m$ 个角度 $\theta_k$ 能让这个多项式等于 0。那这 $m$ 个 $\theta_k$ 算出的 $\cot^2(\theta_k)$ 会不会有重复的呢？
   不会！因为在 $(0, \pi/2)$ 上，$\cot^2\theta$ 是一个**严格单调递减**的函数。不同的 $\theta_k$ 必定对应不同的 $\cot^2(\theta_k)$。
   **这 $m$ 个互不相同的数值 $x_k = \cot^2(\theta_k)$，恰好就是这个 $m$ 次多项式的 $m$ 个根！**

4. **应用韦达定理 (Vieta's formulas) 求和**：
   根据高中的韦达定理，一个多项式所有根的和等于 $-\frac{\text{次高项系数}}{\text{最高项系数}}$。
   在这个多项式中，最高项系数是 $\binom{2m+1}{1}$，次高项（$x^{m-1}$）系数是 $-\binom{2m+1}{3}$。
   
   $$\sum_{k=1}^m \cot^2\left(\frac{k\pi}{2m+1}\right) = -\frac{-\binom{2m+1}{3}}{\binom{2m+1}{1}} = \frac{\frac{(2m+1)(2m)(2m-1)}{3 \times 2 \times 1}}{2m+1} = \frac{m(2m-1)}{3}$$

我们就这样神奇地求出了一个复杂三角函数平方的有限和！我们把这个精确的代数结果记为**引理 1**。

---

## 第三步：搭建桥梁 —— 一个经典的不等式放缩

上一步我们求出了 $\cot^2\theta$ 的和，但这和巴塞尔问题里的 $\frac{1}{n^2}$ 有什么关系呢？我们需要一个不等式把三角函数和代数分式连接起来。

在区间 $x \in (0, \pi/2)$ 内，根据几何意义（单位圆中：三角形面积 < 扇形面积 < 外切三角形面积），有一个非常直观的经典不等式：
$$\sin x < x < \tan x$$

由于在这个区间内三个数都是正数，我们将各项平方，不等号方向不变：
$$\sin^2 x < x^2 < \tan^2 x$$

接着对所有项**取倒数**（注意：对正数取倒数时，原有的不等号必须反向）：
$$\frac{1}{\sin^2 x} > \frac{1}{x^2} > \frac{1}{\tan^2 x}$$

现在，我们利用基础的三角恒等式进行替换：
* 右侧：$\frac{1}{\tan^2 x} = \cot^2 x$
* 左侧：$\frac{1}{\sin^2 x} = \csc^2 x = 1 + \cot^2 x$

替换后，我们理顺不等式的方向，得到了极其关键的桥梁：
$$\cot^2 x < \frac{1}{x^2} < 1 + \cot^2 x$$

---

## 第四步：万事俱备，向着无穷大逼近（夹逼定理）

现在，是将所有零件严丝合缝组装起来的时候了。

我们将第三步不等式中的 $x$ 替换为我们之前找到的根 $\theta_k = \frac{k\pi}{2m+1}$，并对 $k$ 从 $1$ 累加到 $m$：

$$\sum_{k=1}^m \cot^2(\theta_k) < \sum_{k=1}^m \frac{1}{\theta_k^2} < \sum_{k=1}^m \left( 1 + \cot^2(\theta_k) \right)$$

我们来看看中间那一项，展开后它已经包含了我们梦寐以求的平方倒数：
$$\frac{1}{\theta_k^2} = \frac{1}{\left(\frac{k\pi}{2m+1}\right)^2} = \frac{(2m+1)^2}{\pi^2} \cdot \frac{1}{k^2}$$

不等式最右侧的 $\sum_{k=1}^m 1$ 显然等于 $m$。
现在，把我们在第二步求得的**引理 1**的结果 $\frac{m(2m-1)}{3}$ 代入不等式的最左边和最右边，得到：

$$\frac{m(2m-1)}{3} < \frac{(2m+1)^2}{\pi^2} \sum_{k=1}^m \frac{1}{k^2} < m + \frac{m(2m-1)}{3}$$

为了让中间孤立地只剩下 $\sum \frac{1}{k^2}$，我们给不等式三边同时乘以 $\frac{\pi^2}{(2m+1)^2}$：

$$\frac{m(2m-1)\pi^2}{3(2m+1)^2} < \sum_{k=1}^m \frac{1}{k^2} < \frac{m\pi^2}{(2m+1)^2} + \frac{m(2m-1)\pi^2}{3(2m+1)^2}$$

**最后的决战：取极限！**

我们让 $m$ 趋向于无穷大 ($m \to \infty$)，分别计算左右两端的极限：

* **左端极限计算**：
  分子展开：$m(2m-1)\pi^2 = (2m^2 - m)\pi^2$
  分母展开：$3(2m+1)^2 = 3(4m^2 + 4m + 1) = 12m^2 + 12m + 3$
  当 $m \to \infty$ 时，多项式的极限取决于最高次项的比值。
  $$\lim_{m\to\infty} \frac{2m^2\pi^2}{12m^2} = \frac{2\pi^2}{12} = \frac{\pi^2}{6}$$

* **右端极限计算**：
  第一项 $\frac{m\pi^2}{4m^2+4m+1}$，分母的 $m^2$ 增长速度远快于分子的 $m$，所以当 $m \to \infty$ 时，这一项极限为 $0$。
  第二项就是我们刚刚算过的左端，极限同样是 $\frac{\pi^2}{6}$。
  所以右端整体的极限为 $0 + \frac{\pi^2}{6} = \frac{\pi^2}{6}$。

根据微积分中的**夹逼定理 (Squeeze Theorem)**，既然左右两端的不等式屏障都死死地向着同一个数值收缩，那么被夹在中间的无穷级数也别无选择，只能等于这个值：

$$\lim_{m\to\infty} \sum_{k=1}^m \frac{1}{k^2} = \sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$$

**Q.E.D. 证明完毕！**

### 结语

回顾整个过程，你会发现这个证明没有任何“魔法”或者难以理解的突变。它巧妙地利用**复数**拆解了三角函数，利用**多项式**提取了有限项的精确和，最后利用**几何不等式**完成了向无穷大跨越的关键一步。初等方法之美，就隐藏在这些严丝合缝、步步为营的代数推演之中。
