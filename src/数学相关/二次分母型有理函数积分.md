# 二次分母型有理函数积分
> 注：此文档为AI生成，不保证正确性 ———— 2026.3.16

我们要求解的积分是：

$$
\int \frac{1}{Ax^2 + Bx + C} \, dx
$$

其中 $ A, B, C $ 是常数，且 $ A \neq 0 $（否则不是二次函数）。这个积分的结果取决于分母二次式的判别式：

$$
\Delta = B^2 - 4AC
$$

我们将根据 $\Delta$ 的正负和零三种情况分别讨论。

---

## **第一步：配方（Completing the Square）**

先对分母进行配方：

$$
Ax^2 + Bx + C = A\left(x^2 + \frac{B}{A}x\right) + C = A\left[\left(x + \frac{B}{2A}\right)^2 - \left(\frac{B}{2A}\right)^2\right] + C
$$

$$
= A\left(x + \frac{B}{2A}\right)^2 - \frac{B^2}{4A} + C = A\left(x + \frac{B}{2A}\right)^2 + \frac{4AC - B^2}{4A}
$$

令：

$$
u = x + \frac{B}{2A}, \quad \text{则} \quad du = dx
$$

于是积分变为：

$$
\int \frac{1}{A u^2 + \frac{4AC - B^2}{4A}} \, du = \int \frac{1}{A \left( u^2 + \frac{4AC - B^2}{4A^2} \right)} \, du
$$

记：

$$
D = \frac{4AC - B^2}{4A^2} = -\frac{\Delta}{4A^2}
$$

所以积分化为：

$$
\frac{1}{A} \int \frac{1}{u^2 + D} \, du
$$

现在根据 $ D $ 的符号（即 $\Delta$ 的符号）分情况讨论。

---

## **情况一：$\Delta < 0$ （即 $D > 0$）—— 无实根，用反正切**

此时 $ D = \frac{4AC - B^2}{4A^2} > 0 $，设：

$$
k^2 = D = \frac{4AC - B^2}{4A^2} \Rightarrow k = \frac{\sqrt{4AC - B^2}}{2|A|}
$$

但为了简化，我们直接写：

$$
\int \frac{1}{u^2 + k^2} \, du = \frac{1}{k} \arctan\left(\frac{u}{k}\right) + C_0
$$

因此原积分为：

$$
\frac{1}{A} \cdot \frac{1}{k} \arctan\left(\frac{u}{k}\right) + C
$$

代入 $ k = \frac{\sqrt{4AC - B^2}}{2|A|} $，注意这里要小心符号。实际上更简洁的方式是保留表达式中的平方根形式。

由于 $ D = \frac{4AC - B^2}{4A^2} $，所以：

$$
\int \frac{1}{u^2 + D} \, du = \frac{1}{\sqrt{D}} \arctan\left( \frac{u}{\sqrt{D}} \right)
$$

因此：

$$
\frac{1}{A} \cdot \frac{1}{\sqrt{D}} \arctan\left( \frac{u}{\sqrt{D}} \right) + C
$$

代入 $ u = x + \frac{B}{2A} $，$ \sqrt{D} = \frac{\sqrt{4AC - B^2}}{2|A|} $

但为了避免绝对值带来的复杂性，我们可以统一写作：

> 当 $\Delta = B^2 - 4AC < 0$ 时，

$$
\int \frac{1}{Ax^2 + Bx + C} \, dx = \frac{2}{\sqrt{4AC - B^2}} \arctan\left( \frac{2Ax + B}{\sqrt{4AC - B^2}} \right) + C
$$

**推导验证：**

从前面：

$$
\frac{1}{A} \int \frac{1}{u^2 + D} du = \frac{1}{A} \cdot \frac{1}{\sqrt{D}} \arctan\left( \frac{u}{\sqrt{D}} \right)
$$

而 $ \sqrt{D} = \frac{\sqrt{4AC - B^2}}{2|A|} $，但如果我们不考虑绝对值（因为最终结果中会抵消），可以写成：

$$
\frac{1}{A} \cdot \frac{2|A|}{\sqrt{4AC - B^2}} \arctan\left( \frac{2A u}{\sqrt{4AC - B^2}} \right)
$$

注意到 $ u = x + \frac{B}{2A} \Rightarrow 2A u = 2Ax + B $

如果 $ A > 0 $，则 $ |A| = A $，系数为 $ \frac{2A}{A \sqrt{...}} = \frac{2}{\sqrt{...}} $

如果 $ A < 0 $，则 $ |A| = -A $，系数为 $ \frac{2(-A)}{A \sqrt{...}} = -\frac{2}{\sqrt{...}} $，但此时分母 $ Ax^2 + Bx + C $ 开口向下，且因 $\Delta < 0$，整个分母恒负，积分仍可定义，但通常我们希望结果为正系数，所以标准公式一般假设 $ A > 0 $ 或取绝对值处理。

但在大多数教材中，标准答案写成：

$$
\boxed{
\int \frac{1}{Ax^2 + Bx + C} \, dx = \frac{2}{\sqrt{4AC - B^2}} \arctan\left( \frac{2Ax + B}{\sqrt{4AC - B^2}} \right) + C \quad (\text{当 } \Delta < 0)
}
$$

这是最常见的形式，隐含了 $ 4AC - B^2 > 0 $。

---

## **情况二：$\Delta = 0$ —— 重根，用倒数形式**

此时 $ B^2 = 4AC $，分母可写成完全平方：

$$
Ax^2 + Bx + C = A\left(x + \frac{B}{2A}\right)^2
$$

所以积分变为：

$$
\int \frac{1}{A \left(x + \frac{B}{2A}\right)^2} \, dx = \frac{1}{A} \int \frac{1}{\left(x + \frac{B}{2A}\right)^2} \, dx
$$

令 $ u = x + \frac{B}{2A} $，则：

$$
\frac{1}{A} \int u^{-2} \, du = \frac{1}{A} \cdot (-u^{-1}) + C = -\frac{1}{A \left(x + \frac{B}{2A}\right)} + C
$$

化简：

$$
-\frac{1}{A x + \frac{B}{2}} + C = -\frac{2}{2Ax + B} + C
$$

所以：

$$
\boxed{
\int \frac{1}{Ax^2 + Bx + C} \, dx = -\frac{2}{2Ax + B} + C \quad (\text{当 } \Delta = 0)
}
$$

---

## **情况三：$\Delta > 0$ —— 两个不同实根，用部分分式或对数形式**

此时分母有两个不同的实根：

$$
x_{1,2} = \frac{-B \pm \sqrt{B^2 - 4AC}}{2A}
$$

我们可以将分母分解为：

$$
Ax^2 + Bx + C = A(x - x_1)(x - x_2)
$$

然后使用部分分式分解：

$$
\frac{1}{A(x - x_1)(x - x_2)} = \frac{1}{A(x_1 - x_2)} \left( \frac{1}{x - x_1} - \frac{1}{x - x_2} \right)
$$

（因为 $ \frac{1}{(x-a)(x-b)} = \frac{1}{a-b} \left( \frac{1}{x-a} - \frac{1}{x-b} \right) $）

所以积分：

$$
\int \frac{1}{Ax^2 + Bx + C} dx = \frac{1}{A(x_1 - x_2)} \int \left( \frac{1}{x - x_1} - \frac{1}{x - x_2} \right) dx
$$

$$
= \frac{1}{A(x_1 - x_2)} \left( \ln|x - x_1| - \ln|x - x_2| \right) + C
$$

$$
= \frac{1}{A(x_1 - x_2)} \ln \left| \frac{x - x_1}{x - x_2} \right| + C
$$

现在代入 $ x_1 - x_2 = \frac{\sqrt{B^2 - 4AC}}{A} $（因为 $ x_1 - x_2 = \frac{2\sqrt{\Delta}}{2A} = \frac{\sqrt{\Delta}}{A} $？等一下：

实际：

$$
x_1 = \frac{-B + \sqrt{\Delta}}{2A}, \quad x_2 = \frac{-B - \sqrt{\Delta}}{2A}
\Rightarrow x_1 - x_2 = \frac{2\sqrt{\Delta}}{2A} = \frac{\sqrt{\Delta}}{A}
$$

所以：

$$
A(x_1 - x_2) = A \cdot \frac{\sqrt{\Delta}}{A} = \sqrt{\Delta}
$$

因此：

$$
\int \frac{1}{Ax^2 + Bx + C} dx = \frac{1}{\sqrt{\Delta}} \ln \left| \frac{x - x_1}{x - x_2} \right| + C
$$

也可以写成：

$$
\frac{1}{\sqrt{B^2 - 4AC}} \ln \left| \frac{2Ax + B - \sqrt{B^2 - 4AC}}{2Ax + B + \sqrt{B^2 - 4AC}} \right| + C
$$

**为什么？**

因为：

$$
x - x_1 = x - \frac{-B + \sqrt{\Delta}}{2A} = \frac{2Ax + B - \sqrt{\Delta}}{2A}
$$

同理：

$$
x - x_2 = \frac{2Ax + B + \sqrt{\Delta}}{2A}
$$

所以：

$$
\frac{x - x_1}{x - x_2} = \frac{2Ax + B - \sqrt{\Delta}}{2Ax + B + \sqrt{\Delta}}
$$

因此：

$$
\boxed{
\int \frac{1}{Ax^2 + Bx + C} \, dx = \frac{1}{\sqrt{B^2 - 4AC}} \ln \left| \frac{2Ax + B - \sqrt{B^2 - 4AC}}{2Ax + B + \sqrt{B^2 - 4AC}} \right| + C \quad (\text{当 } \Delta > 0)
}
$$

---

## **总结**

综合以上三种情况，我们有：

$$
\int \frac{1}{Ax^2 + Bx + C} \, dx =
\begin{cases}
\displaystyle \frac{2}{\sqrt{4AC - B^2}} \arctan\left( \frac{2Ax + B}{\sqrt{4AC - B^2}} \right) + C & \text{若 } B^2 - 4AC < 0 \\
\\
\displaystyle -\frac{2}{2Ax + B} + C & \text{若 } B^2 - 4AC = 0 \\
\\
\displaystyle \frac{1}{\sqrt{B^2 - 4AC}} \ln \left| \frac{2Ax + B - \sqrt{B^2 - 4AC}}{2Ax + B + \sqrt{B^2 - 4AC}} \right| + C & \text{若 } B^2 - 4AC > 0
\end{cases}
$$
