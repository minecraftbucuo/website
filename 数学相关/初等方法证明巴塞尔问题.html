初等方法证明巴塞尔问题（2026.4.25）

如果你曾经接触过微积分或无穷级数，你大概率听说过大名鼎鼎的巴塞尔问题 (The Basel Problem)。这个问题要求计算所有非零自然数平方的倒数和：

∑n=1∞1n2=1+14+19+116+⋯\sum_{n=1}^{\infty} \frac{1}{n^2} = 1 + \frac{1}{4} + \frac{1}{9} + \frac{1}{16} + \cdots n=1∑∞​n21​=1+41​+91​+161​+⋯

这个问题曾困扰了当时的数学界近半个世纪，直到 1734 年被年轻的莱昂哈德·欧拉 (Leonhard Euler) 绝妙地解出，答案是惊人的 π26\frac{\pi^2}{6}6π2​。欧拉最初的证明充满了天才的直觉，但在当时缺乏严密的收敛性证明（他直接将泰勒展开式与无穷乘积画上了等号）。

今天，我们要学习的是由 Ioannis Papadimitriou 在 1973 年发表的一种严密的初等证明方法。它不需要复分析或傅里叶级数等高深工具，只需要你掌握高中级别的三角函数、复数基础（二项式定理）和多项式韦达定理。

让我们一步步揭开这个精美证明的面纱。

第一步：从一个意想不到的角度切入 —— 复数与三角函数展开

证明的第一步，看似和无穷级数毫无关系。我们需要寻找 sin⁡(nθ)\sin(n\theta)sin(nθ) 与 cot⁡θ\cot\thetacotθ 之间的代数关系。

这里我们需要请出连接复数与三角函数的桥梁——棣莫弗公式 (De Moivre's formula)：

(cos⁡θ+isin⁡θ)n=cos⁡(nθ)+isin⁡(nθ)(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta) (cosθ+isinθ)n=cos(nθ)+isin(nθ)

接下来，我们利用二项式定理将等式的左边 (cos⁡θ+isin⁡θ)n( \cos\theta + i\sin\theta )^n(cosθ+isinθ)n 展开。
二项式定理的形式是 (a+b)n=∑k=0n(nk)an−kbk(a+b)^n = \sum_{k=0}^n \binom{n}{k} a^{n-k}b^k(a+b)n=∑k=0n​(kn​)an−kbk。在这里 a=cos⁡θa = \cos\thetaa=cosθ，b=isin⁡θb = i\sin\thetab=isinθ。

由于等式右边实部是 cos⁡(nθ)\cos(n\theta)cos(nθ)，虚部是 sin⁡(nθ)\sin(n\theta)sin(nθ)，我们只关心等式两边的虚部（即所有包含奇数个 iii 的项，因为 i1=ii^1 = ii1=i，i2=−1i^2 = -1i2=−1（实数），i3=−ii^3 = -ii3=−i，i4=1i^4 = 1i4=1（实数），以此类推，符号会交替出现）：

提取虚部后，我们就得到了 sin⁡(nθ)\sin(n\theta)sin(nθ) 的表达式，注意其中的正负号交替：

sin⁡(nθ)=(n1)cos⁡n−1θsin⁡θ−(n3)cos⁡n−3θsin⁡3θ+(n5)cos⁡n−5θsin⁡5θ−⋯\sin(n\theta) = \binom{n}{1}\cos^{n-1}\theta \sin\theta - \binom{n}{3}\cos^{n-3}\theta \sin^3\theta + \binom{n}{5}\cos^{n-5}\theta \sin^5\theta - \cdots sin(nθ)=(1n​)cosn−1θsinθ−(3n​)cosn−3θsin3θ+(5n​)cosn−5θsin5θ−⋯

为了让这个式子变得更整齐，并且引出我们想要的 cot⁡θ\cot\thetacotθ，我们在等式右侧强行提取一个公因式 sin⁡nθ\sin^n\thetasinnθ。
我们拿第二项举例看看提取后会变成什么样：

cos⁡n−3θsin⁡3θsin⁡nθ=cos⁡n−3θsin⁡n−3θ=(cos⁡θsin⁡θ)n−3=cot⁡n−3θ\frac{\cos^{n-3}\theta \sin^3\theta}{\sin^n\theta} = \frac{\cos^{n-3}\theta}{\sin^{n-3}\theta} = \left(\frac{\cos\theta}{\sin\theta}\right)^{n-3} = \cot^{n-3}\theta sinnθcosn−3θsin3θ​=sinn−3θcosn−3θ​=(sinθcosθ​)n−3=cotn−3θ

按照这个规律，所有项都可以完美转化为 cot⁡θ\cot\thetacotθ 的幂，式子变为了一个极其整齐的乘积形式：

sin⁡(nθ)=sin⁡nθ[(n1)cot⁡n−1θ−(n3)cot⁡n−3θ+(n5)cot⁡n−5θ−⋯ ]\sin(n\theta) = \sin^n\theta \left[ \binom{n}{1}\cot^{n-1}\theta - \binom{n}{3}\cot^{n-3}\theta + \binom{n}{5}\cot^{n-5}\theta - \cdots \right] sin(nθ)=sinnθ[(1n​)cotn−1θ−(3n​)cotn−3θ+(5n​)cotn−5θ−⋯]

这就是我们后续所有推导的基石。

第二步：巧妙构造方程，召唤“韦达定理”

现在，真正的魔法开始了。我们要把上面的三角恒等式变成一个标准的多项式方程。

  1. 设定特殊参数，确保最高次幂为偶数：
    我们令 n=2m+1n = 2m + 1n=2m+1（其中 mmm 为正整数）。这样 n−1=2mn-1 = 2mn−1=2m，括号里的最高次幂就是 cot⁡2mθ\cot^{2m}\thetacot2mθ，恰好是偶数次幂，方便我们稍后换元。

  2. 寻找方程的零点（根）：
    我们要让等式左边的 sin⁡((2m+1)θ)=0\sin((2m+1)\theta) = 0sin((2m+1)θ)=0。
    我们知道，当角度是 π\piπ 的整数倍时，正弦值为 0。即 (2m+1)θ=kπ(2m+1)\theta = k\pi(2m+1)θ=kπ（kkk 为整数）。
    由此，我们可以找到 mmm 个位于区间 (0,π/2)(0, \pi/2)(0,π/2) 内的角度：

    θk=kπ2m+1(k=1,2,…,m)\theta_k = \frac{k\pi}{2m+1} \quad (k = 1, 2, \dots, m) θk​=2m+1kπ​(k=1,2,…,m)

    关键逻辑来了： 在 (0,π/2)(0, \pi/2)(0,π/2) 这个区间内，sin⁡(θk)\sin(\theta_k)sin(θk​) 绝对大于 0。回顾我们第一步得到的公式：左边=sin⁡nθ×[方括号]\text{左边} = \sin^n\theta \times [\text{方括号}]左边=sinnθ×[方括号]。
    既然整体等于 0，且 sin⁡nθk≠0\sin^n\theta_k \neq 0sinnθk​=0，那么唯一的可能就是方括号里的那串式子等于 0！

  3. 构建 mmm 次多项式并换元：
    我们将方括号内的式子看作是关于新变量 xxx 的多项式，令 x=cot⁡2θx = \cot^2\thetax=cot2θ。
    因为 cot⁡n−1θ=cot⁡2mθ=(cot⁡2θ)m=xm\cot^{n-1}\theta = \cot^{2m}\theta = (\cot^2\theta)^m = x^mcotn−1θ=cot2mθ=(cot2θ)m=xm，同样 cot⁡n−3θ=xm−1\cot^{n-3}\theta = x^{m-1}cotn−3θ=xm−1，依此类推。多项式方程变为：

    P(x)=(2m+11)xm−(2m+13)xm−1+⋯=0P(x) = \binom{2m+1}{1}x^m - \binom{2m+1}{3}x^{m-1} + \dots = 0 P(x)=(12m+1​)xm−(32m+1​)xm−1+⋯=0

    我们刚才找到了 mmm 个角度 θk\theta_kθk​ 能让这个多项式等于 0。那这 mmm 个 θk\theta_kθk​ 算出的 cot⁡2(θk)\cot^2(\theta_k)cot2(θk​) 会不会有重复的呢？
    不会！因为在 (0,π/2)(0, \pi/2)(0,π/2) 上，cot⁡2θ\cot^2\thetacot2θ 是一个严格单调递减的函数。不同的 θk\theta_kθk​ 必定对应不同的 cot⁡2(θk)\cot^2(\theta_k)cot2(θk​)。
    这 mmm 个互不相同的数值 xk=cot⁡2(θk)x_k = \cot^2(\theta_k)xk​=cot2(θk​)，恰好就是这个 mmm 次多项式的 mmm 个根！

  4. 应用韦达定理 (Vieta's formulas) 求和：
    根据高中的韦达定理，一个多项式所有根的和等于 −次高项系数最高项系数-\frac{\text{次高项系数}}{\text{最高项系数}}−最高项系数次高项系数​。
    在这个多项式中，最高项系数是 (2m+11)\binom{2m+1}{1}(12m+1​)，次高项（xm−1x^{m-1}xm−1）系数是 −(2m+13)-\binom{2m+1}{3}−(32m+1​)。

    ∑k=1mcot⁡2(kπ2m+1)=−−(2m+13)(2m+11)=(2m+1)(2m)(2m−1)3×2×12m+1=m(2m−1)3\sum_{k=1}^m \cot^2\left(\frac{k\pi}{2m+1}\right) = -\frac{-\binom{2m+1}{3}}{\binom{2m+1}{1}} = \frac{\frac{(2m+1)(2m)(2m-1)}{3 \times 2 \times 1}}{2m+1} = \frac{m(2m-1)}{3} k=1∑m​cot2(2m+1kπ​)=−(12m+1​)−(32m+1​)​=2m+13×2×1(2m+1)(2m)(2m−1)​​=3m(2m−1)​

我们就这样神奇地求出了一个复杂三角函数平方的有限和！我们把这个精确的代数结果记为引理 1。

第三步：搭建桥梁 —— 一个经典的不等式放缩

上一步我们求出了 cot⁡2θ\cot^2\thetacot2θ 的和，但这和巴塞尔问题里的 1n2\frac{1}{n^2}n21​ 有什么关系呢？我们需要一个不等式把三角函数和代数分式连接起来。

在区间 x∈(0,π/2)x \in (0, \pi/2)x∈(0,π/2) 内，根据几何意义（单位圆中：三角形面积 < 扇形面积 < 外切三角形面积），有一个非常直观的经典不等式：

sin⁡x<x<tan⁡x\sin x < x < \tan x sinx<x<tanx

由于在这个区间内三个数都是正数，我们将各项平方，不等号方向不变：

sin⁡2x<x2<tan⁡2x\sin^2 x < x^2 < \tan^2 x sin2x<x2<tan2x

接着对所有项取倒数（注意：对正数取倒数时，原有的不等号必须反向）：

1sin⁡2x>1x2>1tan⁡2x\frac{1}{\sin^2 x} > \frac{1}{x^2} > \frac{1}{\tan^2 x} sin2x1​>x21​>tan2x1​

现在，我们利用基础的三角恒等式进行替换：

  • 右侧：1tan⁡2x=cot⁡2x\frac{1}{\tan^2 x} = \cot^2 xtan2x1​=cot2x
  • 左侧：1sin⁡2x=csc⁡2x=1+cot⁡2x\frac{1}{\sin^2 x} = \csc^2 x = 1 + \cot^2 xsin2x1​=csc2x=1+cot2x

替换后，我们理顺不等式的方向，得到了极其关键的桥梁：

cot⁡2x<1x2<1+cot⁡2x\cot^2 x < \frac{1}{x^2} < 1 + \cot^2 x cot2x<x21​<1+cot2x

第四步：万事俱备，向着无穷大逼近（夹逼定理）

现在，是将所有零件严丝合缝组装起来的时候了。

我们将第三步不等式中的 xxx 替换为我们之前找到的根 θk=kπ2m+1\theta_k = \frac{k\pi}{2m+1}θk​=2m+1kπ​，并对 kkk 从 111 累加到 mmm：

∑k=1mcot⁡2(θk)<∑k=1m1θk2<∑k=1m(1+cot⁡2(θk))\sum_{k=1}^m \cot^2(\theta_k) < \sum_{k=1}^m \frac{1}{\theta_k^2} < \sum_{k=1}^m \left( 1 + \cot^2(\theta_k) \right) k=1∑m​cot2(θk​)<k=1∑m​θk2​1​<k=1∑m​(1+cot2(θk​))

我们来看看中间那一项，展开后它已经包含了我们梦寐以求的平方倒数：

1θk2=1(kπ2m+1)2=(2m+1)2π2⋅1k2\frac{1}{\theta_k^2} = \frac{1}{\left(\frac{k\pi}{2m+1}\right)^2} = \frac{(2m+1)^2}{\pi^2} \cdot \frac{1}{k^2} θk2​1​=(2m+1kπ​)21​=π2(2m+1)2​⋅k21​

不等式最右侧的 ∑k=1m1\sum_{k=1}^m 1∑k=1m​1 显然等于 mmm。
现在，把我们在第二步求得的引理 1的结果 m(2m−1)3\frac{m(2m-1)}{3}3m(2m−1)​ 代入不等式的最左边和最右边，得到：

m(2m−1)3<(2m+1)2π2∑k=1m1k2<m+m(2m−1)3\frac{m(2m-1)}{3} < \frac{(2m+1)^2}{\pi^2} \sum_{k=1}^m \frac{1}{k^2} < m + \frac{m(2m-1)}{3} 3m(2m−1)​<π2(2m+1)2​k=1∑m​k21​<m+3m(2m−1)​

为了让中间孤立地只剩下 ∑1k2\sum \frac{1}{k^2}∑k21​，我们给不等式三边同时乘以 π2(2m+1)2\frac{\pi^2}{(2m+1)^2}(2m+1)2π2​：

m(2m−1)π23(2m+1)2<∑k=1m1k2<mπ2(2m+1)2+m(2m−1)π23(2m+1)2\frac{m(2m-1)\pi^2}{3(2m+1)^2} < \sum_{k=1}^m \frac{1}{k^2} < \frac{m\pi^2}{(2m+1)^2} + \frac{m(2m-1)\pi^2}{3(2m+1)^2} 3(2m+1)2m(2m−1)π2​<k=1∑m​k21​<(2m+1)2mπ2​+3(2m+1)2m(2m−1)π2​

最后的决战：取极限！

我们让 mmm 趋向于无穷大 (m→∞m \to \inftym→∞)，分别计算左右两端的极限：

  • 左端极限计算：
    分子展开：m(2m−1)π2=(2m2−m)π2m(2m-1)\pi^2 = (2m^2 - m)\pi^2m(2m−1)π2=(2m2−m)π2
    分母展开：3(2m+1)2=3(4m2+4m+1)=12m2+12m+33(2m+1)^2 = 3(4m^2 + 4m + 1) = 12m^2 + 12m + 33(2m+1)2=3(4m2+4m+1)=12m2+12m+3
    当 m→∞m \to \inftym→∞ 时，多项式的极限取决于最高次项的比值。

    lim⁡m→∞2m2π212m2=2π212=π26\lim_{m\to\infty} \frac{2m^2\pi^2}{12m^2} = \frac{2\pi^2}{12} = \frac{\pi^2}{6} m→∞lim​12m22m2π2​=122π2​=6π2​

  • 右端极限计算：
    第一项 mπ24m2+4m+1\frac{m\pi^2}{4m^2+4m+1}4m2+4m+1mπ2​，分母的 m2m^2m2 增长速度远快于分子的 mmm，所以当 m→∞m \to \inftym→∞ 时，这一项极限为 000。
    第二项就是我们刚刚算过的左端，极限同样是 π26\frac{\pi^2}{6}6π2​。
    所以右端整体的极限为 0+π26=π260 + \frac{\pi^2}{6} = \frac{\pi^2}{6}0+6π2​=6π2​。

根据微积分中的夹逼定理 (Squeeze Theorem)，既然左右两端的不等式屏障都死死地向着同一个数值收缩，那么被夹在中间的无穷级数也别无选择，只能等于这个值：

lim⁡m→∞∑k=1m1k2=∑n=1∞1n2=π26\lim_{m\to\infty} \sum_{k=1}^m \frac{1}{k^2} = \sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6} m→∞lim​k=1∑m​k21​=n=1∑∞​n21​=6π2​

Q.E.D. 证明完毕！

结语

回顾整个过程，你会发现这个证明没有任何“魔法”或者难以理解的突变。它巧妙地利用复数拆解了三角函数，利用多项式提取了有限项的精确和，最后利用几何不等式完成了向无穷大跨越的关键一步。初等方法之美，就隐藏在这些严丝合缝、步步为营的代数推演之中。
