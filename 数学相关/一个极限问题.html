求解一个极限（2026.4.6）

题目：

lim⁡n→∞n(π4−∑i=1nnn2+i2)\lim_{n \to \infty} n \left( \frac{\pi}{4} - \sum_{i=1}^{n} \frac{n}{n^2 + i^2} \right) n→∞lim​n(4π​−i=1∑n​n2+i2n​)

这是一个非常经典的定积分与黎曼和（Riemann sum）误差分析的极限问题。我们可以通过构造函数、将求和转化为积分并利用泰勒公式（Taylor's formula）展开来求解。

以下是详细的推导过程：

1. 构造函数与定积分

首先，观察求和式，将其提取公因子 1n\frac{1}{n}n1​，构造成黎曼和的形式：

∑i=1nnn2+i2=1n∑i=1n11+(in)2\sum_{i=1}^{n} \frac{n}{n^2 + i^2} = \frac{1}{n} \sum_{i=1}^{n} \frac{1}{1 + (\frac{i}{n})^2} i=1∑n​n2+i2n​=n1​i=1∑n​1+(ni​)21​

令 f(x)=11+x2f(x) = \frac{1}{1+x^2}f(x)=1+x21​，则上述和式即为函数 f(x)f(x)f(x) 在区间 [0,1][0, 1][0,1] 上的右端点黎曼和。
我们知道该函数在 [0,1][0, 1][0,1] 上的定积分刚好等于 π4\frac{\pi}{4}4π​：

∫0111+x2dx=arctan⁡(x)∣01=arctan⁡(1)−arctan⁡(0)=π4\int_0^1 \frac{1}{1+x^2} dx = \arctan(x) \Big|_0^1 = \arctan(1) - \arctan(0) = \frac{\pi}{4} ∫01​1+x21​dx=arctan(x)​01​=arctan(1)−arctan(0)=4π​

因此，原极限可以改写为：

L=lim⁡n→∞n(∫01f(x)dx−1n∑i=1nf(in))L = \lim_{n \to \infty} n \left( \int_0^1 f(x) dx - \frac{1}{n} \sum_{i=1}^{n} f\left(\frac{i}{n}\right) \right) L=n→∞lim​n(∫01​f(x)dx−n1​i=1∑n​f(ni​))

2. 利用泰勒公式分析区间误差

为了精确分析积分与黎曼和之间的差值，我们引入积分上限函数 F(x)=∫0xf(t)dtF(x) = \int_0^x f(t) dtF(x)=∫0x​f(t)dt。
显然有 F′(x)=f(x)F'(x) = f(x)F′(x)=f(x)，且 ∫01f(x)dx=F(1)−F(0)\int_0^1 f(x) dx = F(1) - F(0)∫01​f(x)dx=F(1)−F(0)。

我们可以将总误差拆分到每一个小区间 [i−1n,in][\frac{i-1}{n}, \frac{i}{n}][ni−1​,ni​] 上：

∫01f(x)dx−1n∑i=1nf(in)=∑i=1n[(F(in)−F(i−1n))−1nf(in)]\int_0^1 f(x) dx - \frac{1}{n} \sum_{i=1}^{n} f\left(\frac{i}{n}\right) = \sum_{i=1}^{n} \left[ \left( F\left(\frac{i}{n}\right) - F\left(\frac{i-1}{n}\right) \right) - \frac{1}{n} f\left(\frac{i}{n}\right) \right] ∫01​f(x)dx−n1​i=1∑n​f(ni​)=i=1∑n​[(F(ni​)−F(ni−1​))−n1​f(ni​)]

对 F(i−1n)F\left(\frac{i-1}{n}\right)F(ni−1​) 在 x=inx = \frac{i}{n}x=ni​ 处进行带有拉格朗日余项的泰勒展开，步长为 −1n-\frac{1}{n}−n1​：

F(i−1n)=F(in)+F′(in)(−1n)+F′′(in)2!(−1n)2+F′′′(θi)3!(−1n)3F\left(\frac{i-1}{n}\right) = F\left(\frac{i}{n}\right) + F'\left(\frac{i}{n}\right)\left(-\frac{1}{n}\right) + \frac{F''\left(\frac{i}{n}\right)}{2!}\left(-\frac{1}{n}\right)^2 + \frac{F'''(\theta_i)}{3!}\left(-\frac{1}{n}\right)^3 F(ni−1​)=F(ni​)+F′(ni​)(−n1​)+2!F′′(ni​)​(−n1​)2+3!F′′′(θi​)​(−n1​)3

其中 θi∈(i−1n,in)\theta_i \in (\frac{i-1}{n}, \frac{i}{n})θi​∈(ni−1​,ni​)。

由于 F′(x)=f(x)F'(x) = f(x)F′(x)=f(x)，F′′(x)=f′(x)F''(x) = f'(x)F′′(x)=f′(x)，F′′′(x)=f′′(x)F'''(x) = f''(x)F′′′(x)=f′′(x)，代入上式得：

F(i−1n)=F(in)−1nf(in)+12n2f′(in)−16n3f′′(θi)F\left(\frac{i-1}{n}\right) = F\left(\frac{i}{n}\right) - \frac{1}{n} f\left(\frac{i}{n}\right) + \frac{1}{2n^2} f'\left(\frac{i}{n}\right) - \frac{1}{6n^3} f''(\theta_i) F(ni−1​)=F(ni​)−n1​f(ni​)+2n21​f′(ni​)−6n31​f′′(θi​)

移项整理，得到每个小区间上的误差：

F(in)−F(i−1n)−1nf(in)=−12n2f′(in)+16n3f′′(θi)F\left(\frac{i}{n}\right) - F\left(\frac{i-1}{n}\right) - \frac{1}{n} f\left(\frac{i}{n}\right) = -\frac{1}{2n^2} f'\left(\frac{i}{n}\right) + \frac{1}{6n^3} f''(\theta_i) F(ni​)−F(ni−1​)−n1​f(ni​)=−2n21​f′(ni​)+6n31​f′′(θi​)

3. 求和并计算极限

将上述等式从 i=1i=1i=1 到 nnn 求和，得到总误差：

∫01f(x)dx−1n∑i=1nf(in)=−12n2∑i=1nf′(in)+16n3∑i=1nf′′(θi)\int_0^1 f(x) dx - \frac{1}{n} \sum_{i=1}^{n} f\left(\frac{i}{n}\right) = -\frac{1}{2n^2} \sum_{i=1}^{n} f'\left(\frac{i}{n}\right) + \frac{1}{6n^3} \sum_{i=1}^{n} f''(\theta_i) ∫01​f(x)dx−n1​i=1∑n​f(ni​)=−2n21​i=1∑n​f′(ni​)+6n31​i=1∑n​f′′(θi​)

L=lim⁡n→∞[−12n∑i=1nf′(in)+16n2∑i=1nf′′(θi)]L = \lim_{n \to \infty} \left[ -\frac{1}{2n} \sum_{i=1}^{n} f'\left(\frac{i}{n}\right) + \frac{1}{6n^2} \sum_{i=1}^{n} f''(\theta_i) \right] L=n→∞lim​[−2n1​i=1∑n​f′(ni​)+6n21​i=1∑n​f′′(θi​)]

我们可以将上式分为两部分来求极限：

第一部分：主部极限

lim⁡n→∞−12(1n∑i=1nf′(in))\lim_{n \to \infty} -\frac{1}{2} \left( \frac{1}{n} \sum_{i=1}^{n} f'\left(\frac{i}{n}\right) \right) n→∞lim​−21​(n1​i=1∑n​f′(ni​))

括号内正是函数 f′(x)f'(x)f′(x) 在 [0,1][0, 1][0,1] 上的黎曼和，当 n→∞n \to \inftyn→∞ 时，它趋于定积分 ∫01f′(x)dx\int_0^1 f'(x) dx∫01​f′(x)dx。
由牛顿-莱布尼茨公式，∫01f′(x)dx=f(1)−f(0)\int_0^1 f'(x) dx = f(1) - f(0)∫01​f′(x)dx=f(1)−f(0)。

第二部分：余项极限

lim⁡n→∞16n2∑i=1nf′′(θi)\lim_{n \to \infty} \frac{1}{6n^2} \sum_{i=1}^{n} f''(\theta_i) n→∞lim​6n21​i=1∑n​f′′(θi​)

因为 f(x)=11+x2f(x) = \frac{1}{1+x^2}f(x)=1+x21​ 是初等函数，其二阶导数 f′′(x)f''(x)f′′(x) 在闭区间 [0,1][0, 1][0,1] 上必定连续且有界。设 ∣f′′(x)∣≤M|f''(x)| \le M∣f′′(x)∣≤M，则有：

∣16n2∑i=1nf′′(θi)∣≤16n2∑i=1nM=n⋅M6n2=M6n\left| \frac{1}{6n^2} \sum_{i=1}^{n} f''(\theta_i) \right| \le \frac{1}{6n^2} \sum_{i=1}^{n} M = \frac{n \cdot M}{6n^2} = \frac{M}{6n} ​6n21​i=1∑n​f′′(θi​)​≤6n21​i=1∑n​M=6n2n⋅M​=6nM​

4. 最终结果

综合上述分析，原极限 LLL 为：

L=−12(f(1)−f(0))L = -\frac{1}{2} \Big( f(1) - f(0) \Big) L=−21​(f(1)−f(0))
